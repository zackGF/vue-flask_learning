# vue-flask_learning
自学vue和flask 练习中。。。。


没有任何ui界面加持
代码没有重构看起来会有些冗余

vue程序
```
npm install 
```
```
npm run dev
```

flask使用的是sqlite数据库

安装环境
```
pip install -r requirements.txt
``` 
初始化数据库
```
python manage db init
```
```
python manage db migrate
```
```
python manage db upgrade
```
运行程序
```
python manager.py runserver -r -d
```
